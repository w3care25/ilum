Site Editing

 

Channel

A Channel is where your current content is placed. When you publish new content, a new Channel is automatically created.

Create Tab

Creates new entries in a selected channel.

Edit Tab

Make changes to the content already published to your website.

Once you select an option from the submenu, you will be taken to a listing of existing entries.

On this screen, you can select and remove entries permanently, filter the list down even further, or select an entry to edit. You can also edit common properties and categories in bulk by selecting multiple entries at once.

Publishing / Editing a Channel Entry

Publishing or Editing a Channel Entry is simple.

The Title is the name you want to give the entry.

The URL title is automatically filled out for you, so you don't even have to touch it.

The Summary field is for a brief summary of the page. This is used for SEO meta data, site search engines, and even listing pages.

The Entry Image field is used to select or upload an image to be used to represent this entry. This can be used for SEO Meta Data, site search engines, listing pages, and other possible uses.

Content is a dynamic field that allows to you to select from a number of elements/widgets (can vary from site-to-site) and allows the editor to display custom content in various formats. The most common of these widgets is the Text element.

The Text editor is known as a WYSIWYG (pronounced wiz­-e-­wig). This stands for “what you see is what you get”. Here you can change the text, font, add images, etc. It should look fairly familiar to you if you've used Microsoft Word or a similar publishing program. It even has spell check.

NOTE: When published to the site, the content in the WYSIWYG may appear differently than it does in the editor. For a preview of your site's styles, view your Style Guide page.

Here is a quick overview of several oft-asked about functions of the text editor:

Line Breaks

To start a new paragraph, hit the ENTER key.

To start a new line, hit SHIFT + ENTER (simultaneously).

Change Font, Font Size or Text Color

You may be wondering where these are at? They are disabled. Generally, we want to leave control of these features to the website's Style Sheets. This helps ensure a uniform look across all pages and is more attractive to Search Engines. You can use the Normal and Styles buttons to access predefined styles which may alter font, size and/or color but complete access to these features is restricted. If the Colors button is available, it will be restricted to colors used on the site.

Copying and Pasting Content

If you already have content written in a document, such as Microsoft Word, you can copy and paste it into your content editor with one added step. Once your content has been copied, click on the Paste from Word or Paste as Plain Text button (they look like this:   and paste your content into the window. This removes the spacing and formatting code that is automatically developed from your Word editor.

Add an Image

In the Editor toolbar, click the Image button (  ). You will then be presented with the window you see below. You can paste an image URL here to use an existing image or click the Browse Server button to select an image from your site or upload a new one.

You can use the search box to filter the images you've already uploaded. If you want to upload a new image, click the Upload New File button in the bottom, left-hand corner (you may need to select a directory first if one is not defaulted to) and you will be walked through the process of selecting and uploading a new image.

Image size and links can be setup in the main Image window, as well as custom styles, so play around with it a bit.

Add a Link

First, type the text that you want to turn into a link. Highlight it and then click the Link button (  ) in the toolbar. The window that appears is similar to the Image window. You can paste a link to any website in the URL box (ex: http://www.google.com) or click Browse Server to select a file to link to. You can upload a new file using the Upload New File button (you may need to select a directory first if one is not defaulted to). When you're done, the text you highlighted will link to the file or page you selected.

Other Fields

Some Channel Entry forms will have additional fields for addition content types on their respective pages (ex: the Homepage Sliders field generally found on the Homepage channel). Instructions are provided at the top of each field to help explain what needs to be there and where the content will be used.

Date Tab

Entry Date: This is automatically populated with the current time but you can adjust it to a different time. If you adjust it to a future time and date, that entry will not be displayed on the Website until the current time matches the time of the entry. This is useful for pre-publishing blogs or holiday-themed content. Also, on pages where we list multiple entries as links, those are generally organized by most recent to oldest so you can change the Entry Date in order to affect the order if desired.

Expiration Date: Select a date and time here if you would like the entry to not be displayed on the Website after it "expires". Useful for seasonal content and prevents you from having to remember to take it down.

Categories Tab

NOTE: This tab may not contain any categories on certain channels, as they are not always necessary.

This tab allows you select one or multiple categories to publish the entry to. This is generally used for organizational purposes. Blogs have categories so a user can select a category and see all the other blogs in that category. We also use Main Page and Sub Page categories quite often. This determines if the content in the entry appears on the sections landing page (Main Page) or if it appears on a separate page in that section (Sub Page). Generally, Sub Page is automatically selected for you.

For certain Category Groups, we may give you permission to Add, Edit, Delete and Reorganize categories. Mostly for groups like Blog where you'll be able to add as many categories as you need.

Options Tab

Channel: Allows you to select which channel to publish to. There's almost never any instance this feature is used.

Status: An “Open” entry means that it will display. A “Closed” entry means it will not display. This is useful when putting up temporary content. You can also set a date for it to expire under the Date tab.

Author: Let's you choose who is designated as the Author of the Entry. This is mostly used with our Blog channel to determine who to give the by-line to, that way you can publish an entry for someone else and choose them as the author.

Make Sticky Entry: If you check this box, this Entry will remain at the top of entries pulled regardless of when it was published.

SEO Meta Data Tab


This tab provides you with fields for entering SEO title and description. Search Engines use this information to classify each page on your website for searching purposes. If left blank, we will generate a default set of keywords and descriptions when the page loads, but it's generally preferable to enter your own.

Additionally fields may be available here, such as Robots, which controls how Search Engines access and treat the page, and Ping Search Engines, which when checked will notify Search Engines to re-crawl and index the page. Additional fields may be added, and are generally used by OMG to manage SEO. Descriptions are provided with each field.

Revisions Tab


If you would like to roll back to a previous version of an entry, simply click on the Revisions tab and select Load Revision for the date you’d like to go back to. The old revision isn't saved until you click Save Revision or Submit again, to allow you to view what is in the older revision before committing to it. Generally, we save the last 50 Revisions for each channel entry.

My Profile

By clicking your Screen Name in the top-right corner of the Control Panel and then clicking My Profile in the list that appears, you will be taken your profile editing section.

The most useful tool in this section is Username & Password. This will allow you to change the username that you log in, your Screen Name or your Password.

There are quite a few other options here, but most aren't really used on most sites.
